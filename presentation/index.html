What is SecurityNode?

An embedded security platform that detects physical intrusions and verifies threats using computer vision — running entirely on the edge, with no cloud dependency.

🔒

Sensor-Driven

Door contacts and PIR motion sensors continuously monitor access points

👁️

Vision-Augmented

ESP32-CAM captures and analyzes images to verify if a face is visible or covered

🧩

Modular

Works as a basic alarm alone; vision module adds intelligence without breaking core functionality

The Problem

Traditional alarms make noise, but they cannot tell you what triggered them.

❌ False Alarms

Pets, wind, or authorized users trigger the same siren as an actual intruder

❌ No Context

You know something happened, but not who, when, or whether it was a real threat

✅ The Goal

Local, privacy-aware visual verification before raising an alarm

Edge AI on the ESP32-CAM determines threat level before the alarm sounds.

2

ESP32-C3 sends UART command to ESP32-CAM

3

Image Capture & Analysis

OV2640 captures frame; algorithm checks face visibility

4

Decision

CLEAR → Log & return to idle  |  SUSPICIOUS → Activate alarm

All processing is local. No cloud, no subscription, no privacy risk.

Modularity & Reliability

The system is designed to degrade gracefully. The vision module is an upgrade, not a dependency.

Scenario ESP32-CAM State System Behavior
Full Operation Connected & responsive Vision-assisted alarm mode
Timeout Connected, no response (5s) Fallback to basic alarm
Disconnected Not installed Basic sensor → alarm mode

Power rails remain stable without the CAM: the TPS2553 5V branch simply idles unloaded (no minimum load required), and the AP2112K 3.3V rail gains additional current headroom. No rails are "lost" — the regulator capacity designed for the CAM becomes safety margin for the controller.

🔧 Modular

Add or remove the CAM without changing firmware logic

🛡️ Robust

Watchdog timer, ESD protection, current limiting, soft-start

Conclusion & Future Work

SecurityNode demonstrates how embedded electronics, edge AI, and modular design can create affordable, privacy-first security systems.

✅ What We Built

  • Dual-ESP32 architecture (controller + vision)
  • State-machine firmware with UART protocol
  • Local face-coverage detection
  • Graceful degradation to basic alarm
